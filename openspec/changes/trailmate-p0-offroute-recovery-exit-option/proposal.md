# Change: Off-route recovery safe-exit option

## Why

When a hiker is off route, recovery guidance should not imply that returning to the planned route is the only safe next decision. The user may need to stop, compare exit options, use recorded breadcrumbs, or share location before deciding whether to continue.

## What Changes

- Add a visible "查看安全退出" recovery action for reliable off-route fixes.
- Keep low-accuracy recovery focused on waiting for GPS rather than choosing an exit direction.
- Keep rejoined-route acknowledgement unchanged.

## Non-Goals

- Computing nearest roads or external rescue contacts.
- Changing route-deviation thresholds.
- Making recovery action chips individually clickable.
